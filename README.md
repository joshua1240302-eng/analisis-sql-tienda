# Análisis de Ventas — SQL 🗄️

## Descripción
Análisis de una base de datos de tienda con 4 tablas relacionadas:
clientes, productos, pedidos y detalle de pedidos.

## Hallazgos principales
- CDMX genera el 67% del ingreso total
- Ticket promedio de $6,355 por pedido
- 4 clientes nunca han comprado Electrónica
- Plumas y Monitor lideran ventas en su categoría

## Herramientas usadas
- SQL / SQLite
- SQLiteOnline

## Estructura
- `analisis_tienda.sql` — 10 queries con hallazgos de negocio
